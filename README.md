# VoiceScript 
 A pen plotter that physically writes down whatever you say to it. It's like speech to text, but now its actually writing.

 # Features
 - Primarily using [SpeechRecognition](https://pypi.org/project/SpeechRecognition/) and [Hershey Fonts](https://pypi.org/project/Hershey-Fonts/), my code converts speech into mappable G-code. This G-code is then automatically sent to a plotter, allowing spoken words to be written down on a piece of paper.
 - The pen is held up by a 3D-printed pen holder made in Fusion 360. The holder uses the pens own weight to keep the pen stable, allowing for smooth and non-blotchy text.
